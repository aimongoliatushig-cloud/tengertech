# Role бүтэц

## Үйл ажиллагаа хариуцсан менежер
- Төсөл үүсгэнэ
- Төслийн менежер онооно
- Төсөл батална
- Нийт товч хяналтын самбар харна
- Аль ч тайлан дээр override шийдвэр гаргаж чадна

## Төслийн менежер
- Өөрт оноогдсон төслийг хүлээн авна
- Багийн ахлагч онооно
- Ажил үүсгэнэ
- Эцсийн хугацаа тогтооно
- Тайланг үндсэн шатанд шалгаж батална эсвэл буцаана

## Багийн ахлагч
- Өдөр тутмын ажлыг гүйцэтгэнэ
- Ажлын явц шинэчилнэ
- Тайлан оруулна
- Зураг хавсаргана
- Саад, асуудлыг тэмдэглэнэ

## Ажилтан
- Өнөөдрийн ажил болон сарын хуваарь харна
- Зөвхөн унших эрхтэй

## Эрхийн гол зарчим
- Үйл ажиллагаа хариуцсан менежер: full visibility + override
- Төслийн менежер: өөрийн төслийн write access
- Багийн ахлагч: өөрт оноогдсон ажлын execution access
- Ажилтан: assigned work read-only
