# Workflow

## Төслийн урсгал
1. Ерөнхий менежер төсөл үүсгэнэ
2. Төслийн менежер онооно
3. Төсөл `батлагдаагүй` төлөвт байна
4. Ерөнхий менежер баталснаар төсөл идэвхтэй ашиглагдана

## Ажил хуваарилалт
1. Төслийн менежер багийн ахлагч онооно
2. Төслийн менежер ажил үүсгэнэ
3. Багийн ахлагч болон ажилтнуудыг ажилд холбоно
4. Эцсийн хугацаа тогтооно

## Гүйцэтгэл
1. Багийн ахлагч ажил эхлүүлнэ
2. Явцыг шинэчилнэ
3. Саатвал `саатсан` төлөвт оруулна
4. Дуусвал `дууссан` болгоно

## Тайлангийн урсгал
1. Ноорог
2. Илгээсэн
3. Баталсан эсвэл Буцаасан

## Баталгаажуулалт
- Төслийн менежер бол primary approver
- Ерөнхий менежер хүссэн үедээ override хийнэ

## Mobile урсгал
- Dashboard
- Өнөөдрийн ажил
- Ажлын дэлгэрэнгүй
- Тайлан оруулах
