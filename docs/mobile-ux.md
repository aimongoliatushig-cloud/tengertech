# Mobile UX зарчим

## Ерөнхий зарчим
- Mobile-first
- Нэг дэлгэц дээр нэг гол зорилго
- Том, товч action button
- Card-based layout
- Short form sections
- Trustworthy өнгө ба тодорхой badge

## Screen priority
1. Хяналтын самбар
2. Өнөөдрийн ажил
3. Ажлын дэлгэрэнгүй
4. Тайлан / зураг

## Tap budget
- Өнөөдрийн ажил олох: 1-2 товшилт
- Ажил нээх: 1 товшилт
- Тайлан илгээх: 3 товшилтоос ихгүй

## Ерөнхий менежерийн UX
- KPI stat button
- Хэт дэлгэрэнгүй ERP form харагдахгүй
- Шийдвэр гаргахад хэрэгтэй summary эхэндээ харагдана

## Төслийн менежерийн UX
- Миний төсөл
- Анхаарах ажил
- Дутуу тайлан
- `Ажил нэмэх` primary action

## Багийн ахлагчийн UX
- Өнөөдрийн ажил
- `Тайлан оруулах` primary action
- Зураг хавсаргах хэсэг form-ийн доод хэсэгт
- Sticky bottom action bar ашиглана

## Ажилтны UX
- Өнөөдрийн ажил
- Сарын хуваарь
- Read-only, будилах action байхгүй
